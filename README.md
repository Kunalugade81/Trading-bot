Binance Demo Futures Trading Bot
A Python-based automated trading bot built for Binance Futures (Demo/Testnet environment) using the official Binance API.
This bot allows users to place Market and Limit orders from a command-line interface with logging and proper error handling.

Features

✔ Supports Market and Limit orders

✔ Works with Binance Demo Futures API

✔ Supports BUY and SELL sides

✔ Command-line interactive mode

✔ Error handling for Binance error codes (-2015, -2010, etc.)

✔ Logging of activity and errors

✔ Reusable and clean Python class structure
